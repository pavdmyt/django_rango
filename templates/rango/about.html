Rango About page

This website is a result of marvelous journey through How To Tango With Django 1.7 book.

I used Python for approximately 2 years and built some tools to automate certain daily tasks. The problem was that I couldn't provide these tools to my colleagues. Every time I was asked to launch this or that script and send back generated results. I decided to learn Django because it provides powers of platfom-independent web-browser to deliver app functionality to the end users. I tried How To Tango With Django which sufficiently covered my needs. During approx 2 months of working through tutorial I have learned a whole lot of new concepts, ideas, technologies and approaches. I think it's a great place to start learning web development in Python.

I would like to say Thank You to creators and contributors of How To Tango With Django book.
